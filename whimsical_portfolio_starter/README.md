## Development

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deployment

### GitHub Pages
Update your `vite.config.js`:
```js
base: '/your-repo-name/'
```

Then build:
```bash
npm run build
```

Push the `dist/` folder to GitHub Pages (gh-pages branch or via Settings > Pages).

Or deploy directly with:
- [Vercel](https://vercel.com/)
- [Netlify](https://netlify.com/)
